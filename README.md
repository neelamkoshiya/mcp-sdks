# MCP SDK Examples

This repository contains minimal examples for Model Context Protocol (MCP) SDKs across different programming languages.

## Available SDKs

- **Python** - Feature-complete SDK for ML workflows and prototyping
- **TypeScript/JavaScript** - Web and Node.js integration via npm
- **Java** - Scalable, thread-safe enterprise integration
- **C#/.NET** - Native .NET integration for cloud and desktop apps
- **Go** - Lightweight, high-performance microservices and CLI tools

## Getting Started

Each SDK directory contains:
- `README.md` - Setup and usage instructions
- Basic MCP server example
- Simple client example (where applicable)

Navigate to each SDK directory for specific instructions.
